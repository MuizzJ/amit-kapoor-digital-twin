Self-Hosting Option

Umbrel Integration — Your Data, Your Server

🏠
What is Umbrel?
An open-source personal server OS — apps as Docker containers on your own hardware. Think of it as a private App Store. Runs on a NUC, Raspberry Pi, or any Linux box.
🔐
Why self-host?
All conversation logs and user data stay on premises. No cloud provider sees the queries. API calls to Claude, Pinecone, and ElevenLabs still go out — but the orchestration and logs are local.
💰
Cost
One-time hardware (~$150–300). API costs identical. Makes sense for air-gapped operation or if you later want to replace Claude with a local LLM (Ollama) for fully offline use.

How It Was Built

The Tech Stack — What Each Tool Does

🤖
Claude Sonnet 4.6
The Brain
Anthropic's AI model — reads Amit's retrieved passages and synthesises a fresh, first-person answer in his voice. Handles streaming, prompt caching, and tool use. Never invents — only speaks from evidence.
🧩
Pinecone
The Memory
A vector database that stores every chunk of Amit's books and essays as a mathematical fingerprint. When a question arrives, Pinecone finds the 8–10 most relevant passages in milliseconds using semantic (meaning-based) search, not keyword matching.
🎙
ElevenLabs
The Voice
Cloned Amit's voice from a recording. Converts the AI-generated text to audio using that cloned voice model in real time. The "Listen" button on every answer streams this audio directly to the browser.
🔍
Tavily
The News Feed
A web search API purpose-built for AI. When the question mentions current events (crises, budgets, recent data), Tavily fetches live web results. Claude then grounds Amit's frameworks in what is actually happening today.
▲
Vercel
The Host
Hosts the entire website globally — auto-deploys from GitHub, scales to any traffic, and handles streaming responses. Every git push triggers a new production deployment in ~30 seconds. Free tier covers all current usage.
⚡
Next.js + React
The Framework
The web framework running the site. It handles routing, the chat widget, the team dashboard, and the API routes that orchestrate all the other tools. The streaming typewriter effect and source chips are all built in React.
Total cost to run: ~$0.003 per cached question · ~$0.02 cold · ElevenLabs ~$0.002/response · Vercel free tier · Pinecone free tier
